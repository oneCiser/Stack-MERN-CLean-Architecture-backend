# MERN  API rest Clean Architecture: Example 1
API rest using clean architecture and MERN stack

## APIS
The following url is used to access the different resources **/api/v1/**
### **Auth** /auth
- POST **/signup**
- POST **/login**
- POST **/logout**

### **Products** /product
- POST **/create**
- GET **/:id**
- GET **/list**
- PUT **/:id**
- DELETE **/:id**
- PUT **/:id/photo**
